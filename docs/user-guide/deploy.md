# High Side Deployment
### On the private registry node

  1. Acquire root
```
 sudo -i
```
  2. Assign variables
```
 vi answer.sh
```
  3. Run [Konductor]
```
 ./konductor.sh
```
    
[CloudCtl]:https://github.com/CodeSparta/CloudCtl
[Konductor]:https://github.com/CodeSparta/Konductor
