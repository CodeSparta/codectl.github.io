# Deploy
## RedHat OpenShift 4.5+ | AirGaped | High Side Infrastructure Deployment
### [CloudCtl] + [Konductor]    
  1. Acquire Root
```
 sudo -i
```
  2. CloudCtl Standup & Run Konductor
```
 ./start-cloudctl.sh
```
  3. Answer "Yes" when prompted to deploy cluster.
